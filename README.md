## ts-starter

A typescript template

## Try it now!

```bash
$ pnpm create un
# or
$ pnpm create un [my-project] -t ts-starter
```

Init git hook(optional)

```shell
$ pnpm git-hook-init
```

## Development

Run `src/index.ts`

```shell
$ pnpm start
```

## Test

```bash
$ pnpm test
```

## Build

Build your lib through `unbuild`

```shell
$ pnpm build
```

## Publish
